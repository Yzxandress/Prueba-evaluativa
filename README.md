#  To do List App - Prueba Evaluativa

## Descripción
Aplicación web para gestionar tareas (CRUD completo), desarrollada con React y Node.js.

## Tecnologías
- Frontend: React
- Backend: Node.js + Express
- Base de datos: PostgreSQL

## Funcionalidades
- Crear tareas
- Listar tareas
- Editar tareas
- Eliminar tareas
- Filtrar por estado (completadas / pendientes)

##  Cómo ejecutar

### Backend
1. cd backend
2. npm install
3. npm run dev

### Frontend
1. cd frontend
2. npm install
3. npm run dev
